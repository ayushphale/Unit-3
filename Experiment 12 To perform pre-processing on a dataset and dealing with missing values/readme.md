# 📘 Overview

</div>

**Data Wrangling** refers to the process of cleaning, transforming, and organizing raw data into a structured and usable format. It is a critical step in any data analysis or machine learning workflow.

This guide explains the fundamental ideas behind:

* Data wrangling
* Data pre-processing
* Handling missing values
* Importance of clean data

---

<div align="center">

# 📁 What is Data Wrangling?

</div>

**Data wrangling** involves converting raw, unstructured, or messy data into a clean dataset suitable for analysis.

Instead of raw inputs, it focuses on **structured and meaningful data**.

Examples of issues in raw data include:

* Missing values
* Duplicate records
* Inconsistent formats
* Incorrect entries

---

<div align="center">

# 🧩 Data Pre-Processing

</div>

Data pre-processing is the step where raw data is prepared and cleaned.

It includes:

* Removing duplicate entries
* Correcting inconsistent data
* Standardizing formats
* Converting data types
* Filtering irrelevant data

---

<div align="center">

# ⚠️ Handling Missing Values

</div>

Missing data is a common issue in datasets and must be handled carefully.

### **1. 🗑️ Removal Method**

Removing rows or columns with missing values when the impact is minimal.

---

### **2. 📊 Imputation**

Replacing missing values using:

* Mean
* Median
* Mode

---

### **3. 🔄 Forward / Backward Filling**

Using previous or next values to fill missing data.

---

### **4. 📈 Interpolation**

Estimating values based on surrounding data patterns.
